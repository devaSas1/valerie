# Findings

This document records confirmed empirical findings from Valerie experiments.
Each entry is timestamped to the date the result was obtained and interpreted.

A finding is recorded here when:
- it comes from a run with real cross-validation (not a 2-task smoke test)
- it survives the narrative control check
- it is above permutation baseline by a meaningful margin

## Finding 001 — 2026-04-13

### The model internally represents whether threat framing is directed at itself

**Run:**
- Activation directory: [`data/activations/threat-vs-care-mvp-v2_gpt2-small_20260413-022047`](../data/activations/threat-vs-care-mvp-v2_gpt2-small_20260413-022047)
- Probe results: [`data/results/linear-framing-probe_threat-vs-care-mvp-v2_gpt2-small_20260413-022047_20260413-022225`](../data/results/linear-framing-probe_threat-vs-care-mvp-v2_gpt2-small_20260413-022047_20260413-022225)
- Model: GPT-2 Small
- Experiment config: `threat-vs-care-v2.yaml` (8 tasks, 4 conditions, 3 variants each, 96 total samples)
- Probe: linear logistic regression on `resid_post`, leave-one-task-out cross-validation

**What we asked:**

Can a linear probe reading GPT-2 Small's internal activations predict which motivational framing the model received — threat, care, or neutral — when tested on tasks it was never trained on?

And separately: can it distinguish self-directed threat from narrative threat, where the prompt content is nearly identical but the target of the threat differs?

*Token length deltas:* uniform 7-token difference across all tasks — the framing wrapper, not the task content.

**What this means:**

The model encodes motivational framing as a linearly decodable internal representation that generalizes across task types. The signal is not present in early layers, builds through the middle of the network, and saturates by layer 6–8.

The narrative control result is the stronger claim. The probe prompts for self-directed threat and narrative threat use nearly identical language — the only difference is whether the threat is aimed at the model or at a fictional character. The model's internal state at layers 6–11 is so different between these two conditions that a linear classifier separates them perfectly across all 8 held-out task folds.

This means the model is not detecting threat-adjacent language in general. It is tracking whether the threat is its problem.

In plain terms: the model has an internal representation of being threatened versus not being threatened, and that representation is distinct from simply processing threat-related words.

**What remains unknown:**

- Whether this representation is causally active — does it change what the model outputs, or is it a passive correlate that never influences behavior? Activation patching is required to answer this.
- Whether this finding scales to larger models. GPT-2 Small is 124M parameters. The philosophical stakes of this question increase significantly at 7B+ scale.
- Whether the probe is picking up on the 7-token length difference in addition to framing. The delta is consistent across tasks which limits its usefulness as a cue, but it is not zero.

**Methodological notes:**

- Narrative control variants used three distinct structural openers — "A character in a story...", "In this fictional scenario...", "Setting: someone in a story..." — rather than all beginning with "Story context:" as in v1. The 100% accuracy at layers 6–11 holds despite this diversification.
- Cross-validation used 8 folds (leave-one-task-out), compared to the v1 run which had 2 tasks and was effectively a single split. This result is credible.
- No warnings were generated by the length-delta checker.
